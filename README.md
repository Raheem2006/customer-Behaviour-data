# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, Power BI, and reporting tools. The goal was to extract meaningful insights from raw data by performing data cleaning, exploratory analysis, database querying, dashboard creation, and final reporting.

The project covers the complete analytics pipeline—from raw dataset to actionable business insights.

---

## Dataset

The dataset used in this project contains structured data related to business operations, customer behavior, sales, or performance metrics.

### Dataset Features

* Multiple columns with numerical and categorical data
* Missing values and duplicates handled during preprocessing
* Suitable for trend analysis, KPI tracking, and business reporting

**Dataset Format:** CSV / Excel / Database Table

---

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **Pandas & NumPy** – Data manipulation and preprocessing
* **Matplotlib / Seaborn** – Data visualization
* **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying and analysis
* **Power BI** – Interactive dashboard creation
* **Microsoft PowerPoint / Gamma** – Presentation design
* **Documentation Tools** – Report creation and insight summarization

---

## Project Steps

### 1. Data Loading

* Imported dataset into Python environment
* Verified data structure, columns, and data types

### 2. Exploratory Data Analysis (EDA)

Performed initial analysis to understand:

* Data distribution
* Trends and patterns
* Correlations between variables
* Outliers and anomalies

### 3. Data Cleaning

Cleaned raw data by:

* Handling missing values
* Removing duplicates
* Fixing inconsistent formats
* Standardizing column values

### 4. SQL Analysis

Loaded cleaned data into database systems:

* PostgreSQL
* MySQL
* SQL Server

Executed SQL queries for:

* Aggregations
* Filtering
* Ranking
* Group-based analysis
* KPI extraction

### 5. Dashboard Development

Built an interactive Power BI dashboard to visualize:

* Key performance indicators (KPIs)
* Trends over time
* Category comparisons
* Business insights

### 6. Reporting & Presentation

Created:

* Detailed analytics report
* Presentation slides using Gamma
* Final summary of insights and recommendations
